# Applications

User-facing applications for the Cost Intelligence Agency platform.

## Submission Mobile App (Dead Drop)

Mobile app for submitting bills and pricing intelligence.

- **Location:** `/apps/submission-mobile/`
- **Platform:** React Native (tentative) or Native iOS/Android
- **Status:** Planning phase

**Features:**
- Camera capture of bills
- On-device OCR and data extraction
- Interactive PII redaction
- Cryptographic signing
- Submission to backend

## Search Mobile App (Lantern)

Mobile app for searching pricing data before negotiating.

- **Location:** `/apps/search-mobile/`
- **Platform:** React Native (tentative) or Native iOS/Android
- **Status:** Planning phase

**Features:**
- Search procedures by code or name
- Filter by location and facility type
- View regional pricing aggregates
- Export/print negotiation data

## Web Interface

Web-based search and browse interface.

- **Location:** `/apps/web/`
- **Platform:** React or similar (TBD)
- **Status:** Planning phase

**Features:**
- Search and browse pricing data
- View aggregate statistics
- Documentation and resources
- Admin/moderation tools (authenticated)
